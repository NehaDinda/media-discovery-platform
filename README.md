# media-discovery-platform
Media Discovery Platform is a responsive web application designed to help users discover and explore a wide variety of entertainment content in one place. The platform brings together popular Hollywood movies, Indian films, TV shows, and K-dramas, making it easier for users to find something new to watch based on their interests.

The website provides an organized and visually appealing interface where users can browse through different categories of media, view movie and show posters, read short descriptions and synopses, and access additional information through "Learn More" links. The goal of the project is to simplify content discovery and offer a seamless browsing experience for entertainment enthusiasts.

Features
Browse movies and TV shows from multiple categories.
Explore Hollywood, Indian, and Korean entertainment content.
View posters, titles, release details, and short descriptions.
Access additional information through dedicated links.
Responsive design that works across desktops, tablets, and mobile devices.
Clean and user-friendly interface built for easy navigation.
Technologies Used
HTML5 – for structuring the web pages.
CSS3 – for styling, layouts, and responsive design.
JavaScript – for adding interactivity and dynamic behavior.
